# GoLearn API - Desenvolvido com Go + Gin Framework

Uma API REST simples desenvolvida em Go utilizando o framework Gin, simulando uma rede social básica de postagens.

## Funcionalidades

- Listar álbuns (tweets simulados)
- Estrutura pronta para evoluir com criação, atualização e remoção

## Como executar

1. Acesse a pasta do projeto:

```bash
cd learn
go build
go run main.go

A API estará disponível em:
http://localhost:3001
